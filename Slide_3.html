AI가 바꾸는 물류 현장: WandaVision YOLO 시스템 소개

기존 CCTV 인프라를 활용한 지능형 자동화 솔루션

기존 CCTV 카메라를 활용한 AI 기반 컨테이너 인식 시스템

WandaVision YOLO란?

YOLO(You Only Look Once) 딥러닝 기술을 기반으로 한 실시간 객체 탐지 시스템으로, NGL Transportation의 컨테이너 물류 환경에 최적화되었습니다.

핵심 기능

CCTV 영상 실시간 분석

기존 설치된 CCTV 카메라 시스템에서 입력되는 영상을 실시간으로 분석하여 추가 장비 투자 없이 구현 가능합니다.

컨테이너 트럭 자동 감지 및 추적

YOLOv8 알고리즘 기반으로 컨테이너 트럭을 정확하게 감지하고 물류 공간 내에서 이동 경로를 실시간 추적합니다.

컨테이너 번호 자동 인식 (OCR)

Google Gemini AI를 활용한 고정밀 OCR 기술로 컨테이너 번호를 자동으로 인식하고 추출합니다.

출입 정보 자동 기록 및 저장

감지된 컨테이너 정보, 출입 시간, 위치 데이터를 자동으로 기록하고 JSON 형식으로 저장하여 데이터 활용성을 높입니다.

기대 효과: 물류 관리 효율성 증대 및 운영 비용 절감

인력 의존도 감소, 실시간 물류 가시성 확보, 데이터 기반 의사결정 지원으로 물류 운영 최적화
